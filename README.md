# Raid Manager

Manager wow players for one raid :D


## To developer

- [Install vagrant](https://www.vagrantup.com/docs/installation/)
- create one fork
- clone project
```
    git clone https://github.com/<githubuser>/raidmanager.git
```
- start vagrant
```
    vagrant up
```
- go to vagrant project
```
    vagrant ssh
    workon env
    cd work_home
```

- start to developer :D
